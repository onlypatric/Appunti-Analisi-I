## Esempio 1 (somma dei primi $n$ naturali)

**Teoria usata.** [`01-concetti-di-base/1.6-induzione.md`](../../01-concetti-di-base/1.6-induzione.md) (induzione).

Mostrare che per ogni $n\in N$:
```math
\sum_{k=1}^{n} k = \frac{n(n+1)}{2}.
```

**Soluzione (schema).**

- Base $n=1$: $1=\frac{1\cdot 2}{2}$.
- Passo: supponi vero per $n$, allora
  ```math
  \sum_{k=1}^{n+1}k=\left(\sum_{k=1}^{n}k\right)+(n+1)=\frac{n(n+1)}{2}+(n+1)=\frac{(n+1)(n+2)}{2}.
  ```

**Teoria usata.** [`01-concetti-di-base/1.6-induzione.md`](../../01-concetti-di-base/1.6-induzione.md) (induzione).

```math
1+q+q^2+\dots+q^n=\frac{q^{n+1}-1}{q-1}\qquad (q\ne 1).
```

## Esempio 3 (disuguaglianza)

**Teoria usata.** [`01-concetti-di-base/1.6-induzione.md`](../../01-concetti-di-base/1.6-induzione.md) (induzione).

Mostra che $2^n\ge n+1$ per ogni $n\in N$.

**Soluzione (idea).**

- Base $n=0$: $1\ge 1$.
- Passo: se $2^n\ge n+1$, allora
  ```math
  2^{n+1}=2\cdot 2^n\ge 2(n+1)\ge (n+1)+1
  ```
  perché $2(n+1)\ge n+2$.
