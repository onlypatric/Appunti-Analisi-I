## Esempio 1 (somma, prodotto, coniugato)

**Teoria usata.** [`01-concetti-di-base/1.5-3-numeri-complessi.md`](../../01-concetti-di-base/1.5-3-numeri-complessi.md) (forma algebrica, coniugato, prodotto).

Sia $z_1=1+2i$ e $z_2=3-i$.

```math
z_1+z_2=(1+3)+i(2-1)=4+i.
```
```math
z_1z_2=(1+2i)(3-i)=3-i+6i-2i^2=5+5i.
```
```math
\overline{z_1}=1-2i,\qquad |z_1|=\sqrt{1^2+2^2}=\sqrt5.
```

## Esempio 2 (divisione)

**Teoria usata.** [`01-concetti-di-base/1.5-3-numeri-complessi.md`](../../01-concetti-di-base/1.5-3-numeri-complessi.md) (quoziente con coniugato).

Calcola $(1+i)/(2-i)$.

**Soluzione.**
```math
\frac{1+i}{2-i}=\frac{(1+i)(2+i)}{(2-i)(2+i)}=\frac{2+i+2i+i^2}{4+1}=\frac{1+3i}{5}.
```

## Esempio 3 (forma polare)

**Teoria usata.** [`01-concetti-di-base/1.5-3-numeri-complessi.md`](../../01-concetti-di-base/1.5-3-numeri-complessi.md) (forma polare).

Sia $z=1+i$.

```math
|z|=\sqrt2,\qquad \arg(z)=\frac{\pi}{4},
```
quindi
```math
z=\sqrt2\,e^{i\pi/4}.
```

## Esempio 4 (De Moivre)

**Teoria usata.** [`01-concetti-di-base/1.5-3-numeri-complessi.md`](../../01-concetti-di-base/1.5-3-numeri-complessi.md) (De Moivre).

Calcola $(\cos\theta+i\sin\theta)^3$.

**Soluzione.**
```math
(\cos\theta+i\sin\theta)^3=\cos(3\theta)+i\sin(3\theta).
```
