# 1.4 Funzioni

## Definizione

Una **funzione** è una relazione speciale che associa **a ogni** $a\in A$ **esattamente un** $b\in B$.

Scriviamo:
```math
f\colon A\to B,\qquad a\longmapsto f(a).
```

- $A$ si chiama **dominio** di $f$.
- $B$ è il **codominio**.
- La **legge** di $f$ è il modo in cui si calcola $f(a)$.
- L’**immagine** di $f$ è:
  ```math
  \mathrm{im}(f) = f(A) = \{f(a) : a\in A\}\subseteq B.
  ```
- La **controimmagine** (o **preimmagine**) di un sottoinsieme $C\subseteq B$ è:
  ```math
  f^{-1}(C) = \{a\in A : f(a)\in C\}.
  ```

**Esempio.**  
Consideriamo $f\colon R\to R$, $f(x)=x^2$.  
Allora:

- $f(2)=4$, $f(-2)=4$ (legge quadratica);
- $\mathrm{im}(f)=[0,+\infty)$;
- $f^{-1}(\{4\})=\{-2,2\}$;
- $f^{-1}([0,1])=[-1,1]$.

---

## Restrizione e immagine di un sottoinsieme

Se $D\subseteq A$, la **restrizione** di $f$ a $D$ è la funzione:
```math
f|_D\colon D\to B,\qquad f|_D(x)=f(x).
```

L’**immagine di $D$** è:
```math
f(D)=\{f(x) : x\in D\}.
```

**Esempio.**  
Se $D=[0,1]$ e $f(x)=x^2$, allora $f|_D$ prende valori in $[0,1]$ e $f(D)=[0,1]$.

---

## Tipi di funzioni

Sia $f\colon A\to B$.

### Iniettiva

$f$ è **iniettiva** se valori diversi di $A$ danno valori diversi in $B$:
```math
f(a_1)=f(a_2)\quad\Rightarrow\quad a_1=a_2.
```

### Suriettiva

$f$ è **suriettiva** se ogni $b\in B$ è immagine di qualche $a\in A$:
```math
f(A)=B.
```

### Biiettiva

$f$ è **biiettiva** se è sia iniettiva che suriettiva.
In questo caso esiste la **funzione inversa**:

```math
f^{-1}\colon B\to A,\qquad f^{-1}(b)=\text{l’unico }a\text{ con }f(a)=b.
```

**Esempio.**  
La funzione $g\colon R\to R$, $g(x)=2x+1$ è biiettiva:

- iniettiva perché $2x_1+1=2x_2+1\Rightarrow x_1=x_2$;
- suriettiva perché per ogni $y$ esiste $x=(y-1)/2$ con $g(x)=y$;
- quindi ha inversa $g^{-1}(y)=(y-1)/2$.

---

## Composizione di funzioni

Date $f\colon A\to B$ e $g\colon B\to C$, la **composizione** è

```math
g\circ f\colon A\to C,\qquad (g\circ f)(x)=g(f(x)).
```

**Esempio.**  
Se $f\colon R\to R$, $f(x)=x+1$ e $g(x)=x^2$, allora

- $g\circ f(x)=(x+1)^2$,
- $f\circ g(x)=x^2+1$ (attenzione: generalmente $g\circ f\ne f\circ g$).

---

## Riepilogo

- Funzione $f\colon A\to B$ con dominio $A$, codominio $B$, immagine $f(A)$.
- Controimmagine $f^{-1}(C)$ per $C\subseteq B$.
- Restrizione $f|_D$ e immagine di sottoinsiemi $f(D)$.
- Iniettiva: $f(a_1)=f(a_2)\Rightarrow a_1=a_2$.
- Suriettiva: $f(A)=B$.
- Biiettiva: sia iniettiva che suriettiva, quindi $f^{-1}$ esiste.
- Composizione $(g\circ f)(x)=g(f(x))$; attenzione all’ordine.
