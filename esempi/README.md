# Esempi ed esercizi

Questa cartella contiene esempi svolti ed esercizi (con soluzioni) collegati agli appunti teorici.

## Struttura

- `01-concetti-di-base/`
- `02-successioni/`
- `03-limiti-continuita/`
- `04-calcolo-differenziale/`
- `05-integrali/`
- `06-equazioni-differenziali/`

## Indici rapidi

- Concetti di base: `esempi/01-concetti-di-base/`
- Successioni: `esempi/02-successioni/`
- Limiti e continuità: `esempi/03-limiti-continuita/`
- Calcolo differenziale: `esempi/04-calcolo-differenziale/`
  - Studi di funzione (10 completi con grafici): `esempi/04-calcolo-differenziale/studi-di-funzione/README.md`
- Integrali: `esempi/05-integrali/`
- Equazioni differenziali: `esempi/06-equazioni-differenziali/`

## Come usare questi file

1. Parti dal capitolo teorico.
2. Vai al file di esempi corrispondente (linkato nei capitoli).
3. Prova a svolgere l’esercizio senza guardare la soluzione.
4. Confronta con la soluzione e annota gli errori tipici.

## Grafici

Alcuni esempi rimandano a grafici in `assets/plots/` (SVG generati con `scripts/generate_svgs.py`).
